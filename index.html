Tachyons 4 Modded.

This version of Tachyons 4 has additional support for the following features:

Dark Mode classes prefixed with dm- prefix
Works for colors, backgrounds, borders and hover and some utility classes.
Additional helpers utility classes
// Gaps (responsive variants available)
.g0 {gap: 0}
.g1 {gap: .25rem}
.g2 {gap: .5rem}
.g3 {gap: 1rem}
.g4 {gap: 2rem}
.g5 {gap: 4rem}
.g6 {gap: 8rem}
.g7 {gap: 16rem}

// Filters
.active-dim:active { opacity: .8; transition: opacity .15s ease-out }
.invert { filter: invert(1) }

// Misc
.resize-none { resize: none }

// Darkmode
.dm-no-invert { filter: invert(0) }
.dm-invert { filter: invert(1) }
